## Installation 
## For linux(Debian-based)
```bash
apt update
apt upgrade
apt install git
apt install python3
git clone https://github.com/AdityaSodha/Phisher
cd Phisher
python3 Phisher.py
```
## For Termux
```bash
apt update
apt upgrade
pkg install python3
pkg install git
git clone https://github.com/AdityaSodha/Phisher
cd Phisher
python3 Phisher.py
```
